# 🏥 TriageUBS - Sistema de Triagem Inteligente

O **TriageUBS** é um sistema web responsivo e dinâmico desenvolvido para otimizar e agilizar o fluxo de atendimento em Unidades Básicas de Saúde. O projeto simula a rotina de recepção e enfermagem, automatizando a classificação de risco dos pacientes com base em sintomas informados.

## 🚀 Funcionalidades Principais
* **Classificação de Risco Automatizada:** O sistema analisa as queixas principais e sugere a prioridade (Urgente, Emergência, Pouco Urgente) com base nas diretrizes do Protocolo de Manchester.
* **Dashboard em Tempo Real:** Cards dinâmicos que atualizam instantaneamente o número de pacientes aguardando atendimento por nível de gravidade.
* **Fila de Espera Ativa:** Listagem organizada e visual das entradas recentes com badges coloridos de identificação.
* **Layout 100% Responsivo:** Interface mobile-first projetada para funcionar perfeitamente tanto em desktops quanto em tablets e smartphones.

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estruturação semântica da aplicação.
* **CSS3:** Estilização moderna utilizando CSS Grid, Flexbox e variáveis nativas (organizado com arquivos separados para melhor manutenção da responsividade).
* **JavaScript (ES6):** Lógica de negócios isolada (`triagem.js`) e manipulação dinâmica do DOM para atualização de contadores e tabelas (`main.js`).
